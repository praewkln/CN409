# การสร้าง Material ด้วยโปรแกรม Blender
ในส่วนของเนื้อหาที่เรียนรู้ในการทำ Material ผู้จัดทำได้ทำการเรียนรู้การทำ Material เบื้องต้นเป็นรูปทรงโดนัท ซึ่งเป็นขั้นตอนที่ต่อจาก [การสร้างโมเดลด้วยโปรแกรม Blender](/Model.md) โดยมีวิีธีการทำดังนี้

- กดที่รูปทรงของโดนัท เข้าสู่ Edit mode ทำการเลือกพื้นที่ของ icing ของโดนัท และกด shift D เพื่อทำการแยก object ออกจากกันดังภาพ

![Logo](https://github.com/praewkln/CN409/blob/master/mat1.png?raw=true)

- เข้าสู่ Object mode เลือกส่วน icing ของโดนัท ทำการ add modifier >> Solidify , Subdivision surface ดังภาพ

![Logo](https://github.com/praewkln/CN409/blob/master/mat2.png?raw=true)

- ปรับค่าต่างๆ ดังภาพ

![Logo](https://github.com/praewkln/CN409/blob/master/mat3.png?raw=true)

- ทำการเลือกจุดขอบของ icing โดยทำการซ่อนจุดอื่นๆของ icing ดังภาพ

![Logo](https://github.com/praewkln/CN409/blob/master/mat4.png?raw=true)

- ทำการยืดส่วนของขอบ icing ดังภาพ

![Logo](https://github.com/praewkln/CN409/blob/master/mat5.png?raw=true)

- เมื่อทำการยืดส่วนของขอบ icing เสร็จแล้ว ให้ทำการนำจุดที่ซ่อนไว้ออก ดังภาพ

![Logo](https://github.com/praewkln/CN409/blob/master/mat6.png?raw=true)

- ทำการ Sculpting (ปั้นรูปทรง) เพื่อให้มีความสมจริง ดังภาพ

![Logo](https://github.com/praewkln/CN409/blob/master/mat7.png?raw=true)
